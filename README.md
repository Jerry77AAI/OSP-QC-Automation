# OSP-QC-Automation

AI-powered Quality Control automation for OSP Telecom Construction and Permit Drawings.
Essentia Inc. | Co-CTOs: Claude + Grok | Principal: Jerry77AAI

---

## Project Overview

AI automation framework for QC of Construction Drawings (CD), Permit Drawings (PD),
Fiber/Duct/Drop Schematics, ArcGIS Map Data, AutoCAD Templates, and Client Specs.

  - - .claude/hooks/ — Automated guardrails

      - - src/arcgis/ — GIS spatial validation
        - - src/autocad/ — CAD template compliance
